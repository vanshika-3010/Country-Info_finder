# 🌍 Country Info Finder

A web application that allows users to search for any country and view details like capital, population, and flag.

## Features
- Search country by name
- Displays capital
- Shows population
- Displays country flag

## Technologies Used
- HTML
- CSS
- JavaScript
- REST Countries API
